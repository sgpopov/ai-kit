# GitHub Backend

## Submitting the PRD

Create a GitHub issue with:

```bash
gh issue create \
  --title "<feature name> PRD" \
  --body "<prd content>"
```

### Labels (optional)

If the repo uses labels, apply `prd` or `planning` if they exist.

## Constraints

- Do NOT close or edit any existing issues.
- After creating, print the issue URL so the user can navigate to it.
